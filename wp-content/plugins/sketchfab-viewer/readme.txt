Add a button and a shotcode to the Wordpress editor in order to easily add models from sketchfab.com

== Description ==

This plugin lets you add sketchfab models easily on your WordPress.

First, it add a shortcode to the editor [sketchfab id=""].

Second, a new button appears in the WordPress editor.

Third, you can configure the plugins in the options menu.

== Installation ==

1. Upload sketchfab-viewer folder to the wordpress plugin folder (wp-content/plugins)
2. Activate the plugin through the 'Plugins' menu in WordPress

And that's all.

== Frequently Asked Questions ==

= What is Sketchfab? =

Sketchfab is a 3D objects web viewer. You can check it at sketchfab.com

= Why is this plugin useful? =

As you want to embed some models in your posts you need to get the embed code of each models.
With this plugin all you have to do is to get the id and voila it's embedded. Plus the plugins add a shortcode [sketchfab id=""] which is really useful!

== Screenshots ==

1. You will find the button in the toolbar of the WordPress editor.
2. All you need to input is the id of the model. You'll find it at the end of the url (eg. sketchfab.com/show/id)
3. You can set the dimensions of the viewer in the options :)
